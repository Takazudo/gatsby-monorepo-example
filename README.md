# Multiple Gatsby websites in a repository (monorepo)

## setup

```sh
$ yarn install
```

## shared library

```sh
$ yarn dev:shared
```

## site1

```sh
$ yarn dev:site1
```

## site2

```sh
$ yarn dev:site2
```
